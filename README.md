# 🛒 E-Commerce Web Application

A simple Full Stack E-Commerce Web Application that allows users to browse products, search items, add products to a shopping cart, and view the total cart value.

## 📌 Features

- Product Catalog
- Product Search
- Add to Cart
- Cart Management
- Total Price Calculation
- Backend API using Express.js
- Responsive and Simple User Interface

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/ecommerce-web-application.git
```

### Navigate to Project

```bash
cd ecommerce-web-application
```

### Install Dependencies

```bash
npm install
```

### Run Server

```bash
npm start
```

### Open Browser

```text
http://localhost:3000
```
